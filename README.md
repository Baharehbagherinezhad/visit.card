# Visit Card - کارت ویزیت دیجیتال

یک پروژه ساده و زیبا برای نمایش کارت ویزیت دیجیتال، طراحی‌شده با HTML، CSS و JavaScript.

 ویژگی‌ها

- طراحی ساده و مرتب
- اطلاعات تماس (نام، شماره، ایمیل و ...)
- انیمیشن‌ها یا تعاملات ساده با JavaScript
- قابل استفاده در دستگاه‌های مختلف (responsive)

ساختار پروژه

- `2.html`: فایل اصلی HTML
- `2.css`: فایل استایل‌دهی پروژه
- `2.js`: رفتارهای تعاملی ساده با JavaScript
- پوشه `images`: تصاویر استفاده‌شده در پروژه

 تکنولوژی‌های استفاده‌شده

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)


 ایده‌های توسعه بیشتر (در آینده)

- افزودن لینک به شبکه‌های اجتماعی
- بهینه‌سازی برای موبایل (responsive کامل)
- فرم تماس یا ارسال ایمیل مستقیم
- ذخیره اطلاعات با localStorage یا API

 مجوز

این پروژه برای اهداف آموزشی ساخته شده و بدون محدودیت قابل استفاده و توسعه است.
